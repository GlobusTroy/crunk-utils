# Animation Tutorials

This document provides animation-specific tutorial topics and their documentation paths for use with the `mcp0_godot_docs` `fetch_page` function.

## MCP Tool Usage

Use `mcp0_godot_docs` with the following parameters:
- `action: "fetch_page"` - Get tutorials/guides by documentation path
- `path` - The tutorial path (see list below)
- `section: "full"` - Get complete tutorial content

## Animation Topics
- `/tutorials/animation/introduction.html` - Introduction to the animation features
- `/tutorials/animation/animation_track_types.html` - Animation Track types
- `/tutorials/animation/cutout_animation.html` - Cutout animation
- `/tutorials/animation/2d_skeletons.html` - 2D skeletons
- `/tutorials/animation/animation_tree.html` - Using AnimationTree
- `/tutorials/animation/playing_videos.html` - Playing videos
- `/tutorials/animation/creating_movies.html` - Creating movies
