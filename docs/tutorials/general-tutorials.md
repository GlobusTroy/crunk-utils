# General Tutorials

This document provides general tutorial topics and their documentation paths for use with the `mcp0_godot_docs` `fetch_page` function.

## MCP Tool Usage

Use `mcp0_godot_docs` with the following parameters:
- `action: "fetch_page"` - Get tutorials/guides by documentation path
- `path` - The tutorial path (see list below)
- `section: "full"` - Get complete tutorial content

## General Topics
- `/tutorials/best_practices/index.html` - Best practices
- `/tutorials/editor/index.html` - Editor introduction
- `/tutorials/migrating/index.html` - Migrating to a new version
- `/tutorials/troubleshooting.html` - Troubleshooting
